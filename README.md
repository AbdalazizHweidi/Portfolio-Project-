# React Personal Portfolio

## Overview
This is a personal portfolio website built with React to showcase my background, skills, technologies, projects, and contact information.

## Technologies Used
- React
- JavaScript
- CSS
- Git
- GitHub

## Features
- Responsive design
- Component-based structure
- Projects section
- Contact links
- Clean and modern UI

## How to Run Locally

```bash
npm install
npm run dev
```
